About game:
	This game is old, well-known pong game.
	Game is for 2 people.
	You can have fun while playing with your friend.
	If you reach 4 points, game is over.
	Last results will be shown on the screen.
	Game window will be closed in 4 secs after game finishes.

Libraries in need:
	import turtle
	import random
	import time

How to run:
	You can either run on console/terminal or editor/IDE.
